## Overview

A collection of interactive dashboards and reports built with **Power BI**, showcasing data visualization and business intelligence skills.

### Key Highlights

- **DAX Measures** — Custom calculations for dynamic KPIs, time intelligence, and trend analysis
- **Data Modelling** — Star schema design with optimized relationships for fast query performance
- **Interactive Visuals** — Slicers, drill-throughs, and conditional formatting for intuitive data exploration
- **ETL with Power Query** — Data cleaning and transformation pipelines from multiple sources
